# 📩 Spam Detection Web Application

A Machine Learning-based web application that classifies messages as **Spam** or **Not Spam** using Natural Language Processing.

---

## 🚀 Live Demo
👉(https://spam-classifier-2-6w3o.onrender.com)
---

## 💡 Features
- Real-time spam detection
- Clean and interactive UI
- Message history tracking
- Machine Learning model using TF-IDF + Naive Bayes

---

## 🛠️ Tech Stack
- Python
- Flask
- Scikit-learn
- Pandas
- HTML/CSS

---

## 📊 How it Works
1. Text input is taken from user
2. Converted into numerical features using TF-IDF
3. Model predicts whether message is spam or not
